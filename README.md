# Task S1.08

## Level 1 Exercise 1
**Description**:
Creates a list of words, normalizes accents, and prints only
the words that contain the letter 'o'.

## Level 1 Exercise 2
**Description**:
Filters a list of words by length > 5 and by containing 'o'
after accent normalization, then prints the result.

## Level 1 Exercise 3
**Description**:
Builds a list of month names and prints them using a lambda.

## Level 1 Exercise 4
**Description**:
Prints the list of month names using a method reference.

## Level 1 Exercise 5
**Description**:
Defines a functional interface for Pi, assigns 3.1415 via a
lambda, and prints the value.

## Level 1 Exercise 6
**Description**:
Sorts a list of strings by length (shortest to longest) and
prints the ordered list.

## Level 1 Exercise 7
**Description**:
Sorts a list of strings by length (longest to shortest) and
prints the ordered list.

## Level 1 Exercise 8
**Description**:
Uses a functional interface and a lambda to reverse a string,
then prints the reversed result.

## Level 2 Exercise 1
**Description**:
Filters a list of names to those starting with 'A' and with
exactly 3 letters, then prints them.

## Level 2 Exercise 2
**Description**:
Maps integers to "e" or "o" prefixes based on parity, joins
them with commas, and prints the final string.

## Level 2 Exercise 3
**Description**:
Creates an Operation functional interface and applies sum,
subtraction, multiplication, and division via lambdas, then
prints each result.

## Level 2 Exercise 4
**Description**:
Sorts strings by first character and by containing "e", replaces
'a' with '4', prints the list, and then prints only numeric
elements.

## Level 3 Exercise 1
**Description**:
Defines a Student class, fills a list with 10 students, prints
name and age, filters names starting with 'a' into another list,
prints students with grade >= 5, prints those with grade >= 5 and
not in PHP, and prints adults in JAVA.

## Technologies
- Backend: Java - Maven

## Installation and Execution
1. Clone the repository: `git clone https://github.com/Quint3in/Tasca_S1.08.git`
2. Go into the project folder: `cd Tasca_S1.08`
3. Build the module you want (from the root). Example for `n2exercici1`:
   `mvn -pl n2exercici1 -am package`
4. Run it (from the root):
   `java -cp n2exercici1/target/classes org.example.Main`
